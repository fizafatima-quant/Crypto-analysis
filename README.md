\## 🛠️ Setup Guide

1\. Clone repository:

&nbsp;  ```bash

&nbsp;  git clone https://github.com/yourusername/crypto-analysis.git

&nbsp;  ```

2\. Install dependencies:

&nbsp;  ```bash

\## 🚀 How to Run

```bash

\# Calculate indicators
```
